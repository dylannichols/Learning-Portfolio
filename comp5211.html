Course Aim

The aim is to have the skills to publish a web application, produce user documentation and training material.

Learning Outcomes

  1. Migrate a web solution from development to a testing environment.
  2. Create and use a test plan for a web solution.
  3. Move a solution from a test environment to a live platform
  4. Configure applications to meet requirements and obtain client acceptances,
  5. Produce user training material.
